# core

Workspace produit `core` (stable).

Ce produit regroupe les composants coeur de la plateforme:

- `engine`: backend principal d'orchestration/routage.
- `central_ui`: interface centrale de pilotage.
- `launcher`: demarrage/supervision des services.
- `watcher`: surveillance et health checks.

Chaque crate dispose de sa documentation locale dans son propre dossier.
