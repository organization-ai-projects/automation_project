# `pjson_proc_macros`

**Description** : Crate Rust pour les macros procédurales liées à `pjson`.

## Fonctionnalités

- Génération automatique de code.
- Validation des structures JSON.

## Installation

Ajoutez ceci à votre `Cargo.toml` :

