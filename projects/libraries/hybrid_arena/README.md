# `hybrid_arena`

**Description**: Rust crate for managing hybrid arenas.

## Features

- Bump allocation.
- Slot management with generation.

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
hybrid_arena = "0.1.0"
```
