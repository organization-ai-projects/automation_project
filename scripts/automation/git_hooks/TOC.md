## Hooks

- `commit-msg`: Generated by `versioning_automation automation install-hooks`, validates commit message format
- `pre-commit`: Generated by `versioning_automation automation install-hooks`, runs pre-commit checks via Rust CLI
- `pre-push`: Generated by `versioning_automation automation install-hooks`, runs pre-push checks via Rust CLI
- `post-checkout`: Generated by `versioning_automation automation install-hooks`, runs post-checkout checks via Rust CLI
- `versioning_automation automation install-hooks`: Installs git hooks to `.git/hooks/`
